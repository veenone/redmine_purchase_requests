# Features Overview: Redmine Purchase Requests Plugin

## Core Features

### 1. Purchase Request Management
- Create, edit, and track purchase requests
- Multi-stage workflow with configurable statuses
- Detailed request tracking and history

### 2. Budget Management
#### CAPEX (Capital Expenditure)
- Track long-term asset investments
- Quarterly and annual budget allocation
- Capital expense forecasting

#### OPEX (Operational Expenditure)
- Monitor ongoing operational costs
- Granular expense categorization
- Real-time budget utilization tracking

### 3. Vendor Management
- Centralized vendor database
- Vendor performance scoring
- Quote comparison tools
- Vendor communication tracking

### 4. Multi-Currency Support
- Dynamic exchange rate integration
- Multi-currency transaction support
- Automatic currency conversion
- Historical exchange rate tracking

## Advanced Features

### 5. Workflow Customization
- Configurable approval hierarchies
- Role-based access control
- Flexible status transition rules
- Automated approval routing

### 6. Reporting and Analytics
- Comprehensive dashboard
- Customizable financial reports

- Exportable report formats (PDF, CSV)

### 7. Notification System

### 8. Integration Capabilities
- RESTful API for external system integration
