# Redmine Purchase Requests Plugin

## Overview

The Redmine Purchase Requests Plugin is a comprehensive solution for managing purchase requests, budget tracking, and vendor management within Redmine. This plugin provides organizations with a robust tool for controlling and monitoring procurement processes.

## Key Features

- Purchase request creation and management
- Multi-currency support with dynamic exchange rates
- CAPEX and OPEX budget tracking
- Quarterly budget allocation
- Comprehensive vendor management
- Flexible status workflow
- Advanced dashboard and reporting
- Granular permission and access control
- Seamless import/export functionality
- Automated email notifications
